# Flowbit AOI - Frontend Engineer Internship (Submission)

## What this is
Single-page React + TypeScript app that displays a WMS satellite/drone layer, allows drawing AOIs (Area of Interest), toggling the WMS layer, and persists drawn AOIs in `localStorage`. Uses Leaflet (via react-leaflet) and Leaflet Draw for drawing.
